This is the **mobile version of the main homepage** for my personal resume website, optimized for small screens and mobile devices.

## Features

- **Mobile-first design**: Layout designed primarily for smartphones and tablets.
- **Dark/Light mode toggle**: Users can switch between dark and light themes.
- **Hamburger menu navigation**: Collapsible menu for easy access to all sections.
- **Project showcase**: Projects displayed in a vertical scroll-friendly layout with live links to Behance.
- **Animated skill bars**: Skills animate as the user scrolls into view.
- **Contact form**: Mobile-friendly form with placeholder alert for demo purposes.

## Technologies Used

- **HTML5** for structure.
- **CSS3 & custom properties** for styling and theme management.
- **JavaScript** for interactions and animations.
- **Google Fonts (Poppins)** for typography.

## Design Decisions

- **Vertical layout** for easy scrolling and readability on small screens.
- **Cards and sections** to separate experiences, projects, and skills clearly.
- **Gradient accents** to highlight important elements.
- **Accessibility considerations**: Large touch targets, aria labels, and keyboard navigation support.
